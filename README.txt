Very simple application I wrote both to teach myself the basics of OpenGL and
Glut, and as a present to entertain my 5-year-old son who has an obsession with
fans.

Controls:
=========
    Home      - slowly moves the fan
    End       - slowly moves the fan in reverse
    Page Up   - turns the fan on or increases speed
    Page Down - decreases speed or tuns fan off
    +/-       - increases/decreases blade width
    Alt +/-   - increase/decrease blade length
    Mouse     - can be used to click on color pallet to change fan color
    2-6       - adjusts the number of blades in the fan
    g         - changes the type of grate/grill

Compilation:
============
Tested with MinGW, Microsoft Visual C++, and g++ but hopefully works with others
Requires GLUT or FreeGLUT (and of course OpenGL)

http://freeglut.sourceforge.net
